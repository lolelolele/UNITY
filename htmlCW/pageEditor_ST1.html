Splash Screen

For the splash screen, I have decided to go for more of an animated approach, allowing the user to experience a premium feel when entering the webpage. To enhance the user experience, I added a 7-second timed screen to move onto the homepage automatically. Most animations were achieved using CSS, giving the page a dynamic and engaging appearance. Additionally, I incorporated a logo on the splash screen because it represents the mission and identity of the website, adding a touch of brand personality right from the start.

Shop Page

For the shop page, I needed to ensure the user is able to browse multiple items, deciding on 9 items to sell. CSS and JavaScript were used to add user interactivity, enabling features such as adding multiple items to the basket, removing them, adding multiple quantities, and proceeding to checkout. This combination of languages enhances the user experience, providing a seamless shopping journey.

Content Page STUDENT 1

For my content page, I aimed to highlight the objectives and aspirations of our website. I intentionally styled the page to ensure all text is centrally aligned, offering a polished presentation. Additionally, I incorporated an image to complement the content and included a comprehensive "Contact Us" section at the bottom. This section features essential contact details such as an email address, phone number, and physical address, enriching the overall user experience by fostering easy communication and engagement.
